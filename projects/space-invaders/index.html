About this Project

Space Invaders: Enhanced Edition is a modern reimagining of the classic arcade shooter, built entirely with vanilla JavaScript and HTML5 Canvas. This project demonstrates advanced game development concepts including:

  • Custom Game Engine: Built from scratch with a dedicated game loop, state management, and entity-component system.
  • Procedural Generation: Dynamic starfields, nebulas, and particle effects that create a unique visual experience every time.
  • Complex AI Behaviors: Multi-phase boss fights with adaptive attack patterns and pathfinding algorithms.
  • Performance Optimization: Object pooling for particles and projectiles to maintain 60 FPS even with hundreds of entities.

The goal was to create a visually stunning and responsive web game without relying on external game engines or libraries.
