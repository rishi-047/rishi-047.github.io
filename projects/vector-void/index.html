About this Project

Vector Void is a high-octane space shooter that pays homage to the vector graphics era of arcade gaming. It features a custom-built physics engine handling inertia, momentum, and gravity.

  • Vector Physics Engine: Realistic drift mechanics and collision detection built from scratch.
  • Gravity Systems: Dynamic black holes that bend projectiles and pull entities using vector field math.
  • Procedural Generation: Asteroids with unique geometry and splitting mechanics.
  • Procedural Audio: Sound effects generated in real-time using the Web Audio API (oscillators and gain nodes).

The project explores the intersection of math and art, using raw geometry and neon aesthetics to create an immersive experience.
